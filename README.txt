This program was tested on the engr FLIP server.

Input and Output:
1. Run the make command
2. Run the program by calling: tsp [input_file]
   - If both arguments (program executable and input file name) are not provided, an error
   	will be displayed with the correct usage.
   - The input file must be in the same working directory as the program source code file.
3. Results will be printed to [input_file]tour.txt found in the current working directory
